## Data Aggregation

### Existing Bathymetry

Compile existing bathymetry data from [GMRT](https://www.gmrt.org) using the [GMRT MapTool](https://www.gmrt.org/GMRTMapTool/) or from [GeoMapApp](https://www.geomapapp.org/)

Direct downloads are available programmatically using the [GMRT Webservices](https://www.gmrt.org/services/index.php) as NetCDF grid files, ESRI ASCII, or GeoTIFF. See the [wgetBathymetry notebook](../tools/wgetBathymetry.md) for an example.
