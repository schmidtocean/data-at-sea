# Data At Sea

This site is currently under development.

A community-driven resource for managing ocean science data across the expedition lifecycle.

## About

Prioritizing data and data management is a core tenant of good ocean science. This wiki exists to support the community in achieving excellence in data stewardship from planning through publication.

## Expedition Lifecycle

### Pre-Cruise Planning
Data management planning, training resources, and funding guidance for expedition preparation.

### At-Sea Operations
Operational tools, data collection best practices, and controlled vocabularies for shipboard work.

### Post-Cruise Data
QAQC standards, repository submission guidance, and metadata enhancement.

## Contributing

This is a community-driven resource. Contributions, suggestions, and feedback are welcome through GitHub Issues and Pull Requests.
